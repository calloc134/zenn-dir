## 文体ガイド

Zenn の Markdown 記事 (`articles/*.md` / `books/*/*.md`) のリライト・推敲・文体統一を行う際は、以下のスタイルガイドに従ってください。

- `.claude/skills/calloc134-style/SKILL.md` — メインのスタイルガイド
- `.claude/skills/calloc134-style/references/observations.md` — 既存記事から抽出した実例集
- `.claude/skills/calloc134-style/references/vocabulary.md` — 語彙・表記ルールのリファレンス

記事の新規作成・修正・レビュー時は、必ず上記ファイルを読み込み、文体規則を適用してください。
